图书列表

作者: yeguozhong yedaxia.github.com

请求URL

/api/book/list GET

请求参数

参数名 类型 必须 描述
page int 否 页数
limit int 否 每页条数
sort string 否 排序

返回结果

{
  "code": "int",
  "errMsg": "string",
  "data": {
    "total": "int //总记录数",
    "pageCount": "int //页数",
    "currentPage": "int //当前页",
    "pageSize": "int //每页记录数",
    "list": [
      {
        "bookId": "long //图书id",
        "bookName": "string //图书名称"
      }
    ],
    "hasMore": "boolean //是否还有更多"
  }
}

      "userId": "string //用户id",
